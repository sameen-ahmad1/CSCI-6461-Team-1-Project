Part 3 - Instructions

To unpack the jar file run the command “jar xf part3.jar”.

To run the gui/simulator, use the command “java -cp part3.jar gui”. This can also be done on the unpacked files using the command “java gui”.

The source code for program 2 is provided in program2.asm. To run program 2, enter the GUI, type program2.asm in the program file text area, hit IPL, then type program2test.txt in the card reader slot and hit enter, then hit run. When prompted to enter a word, type a word int eh console input field and hit enter for the program to continue.

Test cases for the backend are provided in /testfiles/part3/. In the various sub folders there are tests for the cache, error testing, and for the individual instructions. Individual instructions may also be tested on the gui (pressing IPL with no filepath allows you to set the machine to do so).

The design notes are in readme.md in the folder docs/part3. The git logs are in part3logs.txt in the folder docs/part3.
